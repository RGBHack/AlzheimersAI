Welcome to AlzheimersAI

A web solution for early detection and tracking of Alzheimer's disease

What is AlzheimersAI?

AlzheimersAI is a Firebase-hosted Flask app running inside a Google Cloud Run Docker container. It combines quiz results with a FastAI Convolution Neural Network analysis of the users' brain scan to produce a value for the user's Alzheimers Index. You can view more information about this project along with source code at our GitHub Repository.
